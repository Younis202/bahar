

# منصة تعليمية بحرية - Maritime Learning Platform

## نظرة عامة
منصة تعليمية متخصصة في المجال البحري، مستوحاة من Udemy/Coursera، تدعم محاضرين متعددين من سوريا والأردن، مع حماية فيديو احترافية عبر Bunny Stream، ودعم اللغة العربية (RTL) والإنجليزية.

---

## المرحلة 1: البنية الأساسية والتسجيل

### 1.1 تفعيل Lovable Cloud
- قاعدة بيانات، Authentication، Storage، Edge Functions

### 1.2 نظام التسجيل والدخول
- تسجيل طلاب بالإيميل
- تسجيل محاضرين (يحتاج موافقة الأدمن)
- أدوار المستخدمين: طالب، محاضر، أدمن
- صفحة تسجيل دخول / إنشاء حساب

### 1.3 دعم اللغة العربية والإنجليزية
- واجهة RTL للعربي
- تبديل اللغة في كل صفحة

---

## المرحلة 2: واجهة الطلاب

### 2.1 الصفحة الرئيسية
- بانر ترحيبي وعرض الكورسات المميزة
- تصفح بالتصنيفات (Categories) - ملاحة، هندسة بحرية، سلامة بحرية، إلخ
- شريط بحث عن الكورسات
- فلترة حسب المستوى (مبتدئ/متوسط/متقدم) والسعر والتقييم

### 2.2 صفحة الكورس
- فيديو دعائي (trailer)
- وصف الكورس، ماذا ستتعلم، المتطلبات
- معلومات المحاضر
- منهج الكورس (Topics & Lessons)
- التقييمات والمراجعات
- السعر وزر الشراء

### 2.3 مشاهدة الكورس (بعد الشراء)
- بلاير Bunny Stream مع حماية DRM
- قائمة الدروس الجانبية مع تتبع التقدم
- علامة ✅ على الدروس المكتملة
- شريط تقدم الكورس الإجمالي

### 2.4 الكويزات والاختبارات
- أسئلة اختيار من متعدد
- صح/غلط
- نتائج فورية مع مراجعة الأخطاء
- إعادة الاختبار

### 2.5 الشهادات
- شهادة إتمام تلقائية بعد إنهاء الكورس
- تحميل الشهادة كـ PDF
- اسم الطالب + اسم الكورس + التاريخ

### 2.6 داشبورد الطالب
- كورساتي (اللي اشتريتها)
- تقدمي في كل كورس
- شهاداتي
- إعدادات الحساب

---

## المرحلة 3: داشبورد المحاضرين

### 3.1 إدارة الكورسات
- إنشاء كورس جديد (عنوان، وصف، تصنيف، مستوى، سعر)
- إضافة Topics (وحدات)
- إضافة Lessons تحت كل Topic
- ربط فيديوهات Bunny Stream بالدروس (عبر embed code)
- إضافة مرفقات ومواد تعليمية (PDF, ملفات)
- إنشاء كويزات

### 3.2 إحصائيات المحاضر
- عدد الطلاب المسجلين
- إجمالي الأرباح ونسبة المحاضر
- تقييمات الطلاب
- أكثر الكورسات مبيعاً

---

## المرحلة 4: لوحة تحكم الأدمن

### 4.1 إدارة المستخدمين
- عرض كل المستخدمين (طلاب + محاضرين)
- الموافقة على المحاضرين الجدد أو رفضهم
- تعطيل/تفعيل حسابات

### 4.2 إدارة الكورسات
- مراجعة والموافقة على الكورسات قبل النشر
- إدارة التصنيفات (Categories)

### 4.3 المالية
- تقارير المبيعات
- إعدادات نسب الأرباح (أدمن vs محاضر)
- سجل المدفوعات

### 4.4 إعدادات الموقع
- اللوجو واسم المنصة
- إعدادات الدفع (Stripe)
- إدارة اللغات

---

## المرحلة 5: الدفع والاشتراكات

### 5.1 Stripe Integration
- شراء كورسات فردية
- صفحة checkout آمنة
- فواتير إلكترونية

### 5.2 نظام الكوبونات والخصومات
- أكواد خصم بنسبة أو مبلغ ثابت
- تاريخ انتهاء الكوبون

---

## المرحلة 6: استضافة وحماية الفيديو

### 6.1 تكامل Bunny Stream
- رفع الفيديوهات على Bunny Stream (خارج المنصة)
- تضمين الفيديو في الدروس عبر embed code
- حماية DRM - منع التحميل
- تشغيل الفيديو فقط من دومين المنصة
- جودات متعددة (240p - 4K)
- بلاير مخصص بألوان وشعار المنصة

---

## الملاحظات التقنية
- **Frontend**: React + TypeScript + Tailwind CSS (عبر Lovable)
- **Backend**: Lovable Cloud (Supabase) - Database, Auth, Storage, Edge Functions
- **Video**: Bunny Stream (خارجي) - المحاضر يرفع الفيديو هناك ويضع الكود في المنصة
- **Payment**: Stripe
- **RTL**: دعم كامل للعربي مع تبديل إنجليزي
